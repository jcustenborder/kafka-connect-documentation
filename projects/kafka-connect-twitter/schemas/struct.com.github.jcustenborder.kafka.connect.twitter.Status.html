Status¶

Twitter status message.

Status¶
Name Type Optional Default Value Documentation
CreatedAt INT64 true   Return the created_at
Id INT64 true   Returns the id of the status
Text STRING true   Returns the text of the status
Source STRING true   Returns the source
Truncated BOOLEAN true   Test if the status is truncated
InReplyToStatusId INT64 true   Returns the in_reply_tostatus_id
InReplyToUserId INT64 true   Returns the in_reply_user_id
InReplyToScreenName STRING true   Returns the in_reply_to_screen_name
GeoLocation STRUCT true   Returns The location that this tweet refers to if available.
Place STRUCT true   Returns the place attached to this status
Favorited BOOLEAN true   Test if the status is favorited
Retweeted BOOLEAN true   Test if the status is retweeted
FavoriteCount INT32 true   Indicates approximately how many times this Tweet has been “favorited” by Twitter users.
User STRUCT false   Return the user associated with the status. This can be null if the instance is from User.getStatus().
Retweet BOOLEAN true    
Contributors Array of INT64 false   Returns an array of contributors, or null if no contributor is associated with this status.
RetweetCount INT32 true   Returns the number of times this tweet has been retweeted, or -1 when the tweet was created before this feature was enabled.
RetweetedByMe BOOLEAN true    
CurrentUserRetweetId INT64 true   Returns the authenticating user’s retweet’s id of this tweet, or -1L when the tweet was created before this feature was enabled.
PossiblySensitive BOOLEAN true    
Lang STRING true   Returns the lang of the status text if available.
WithheldInCountries Array of STRING false   Returns the list of country codes where the tweet is withheld
HashtagEntities Array of STRUCT true   Returns an array if hashtag mentioned in the tweet.
UserMentionEntities Array of STRUCT true   Returns an array of user mentions in the tweet.
MediaEntities Array of STRUCT true   Returns an array of MediaEntities if medias are available in the tweet.
SymbolEntities Array of STRUCT true   Returns an array of SymbolEntities if medias are available in the tweet.
URLEntities Array of STRUCT true   Returns an array if URLEntity mentioned in the tweet.
