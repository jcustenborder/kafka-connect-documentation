PointValue¶

This schema represents a value written to InfluxDB.

Fields¶

The name of the value

Optional: yes

Type: STRING

The type of data. This value corresponds to one of fields of this struct.

Optional: yes

Type: STRING

Optional: no

Type: FLOAT64

The int64 value for this field.

Optional: no

The boolean value for this field.

Optional: no

Timestamp value.

Optional: no

Type: INT64

String value.

Optional: no
