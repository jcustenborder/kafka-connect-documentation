Value¶

This schema is used to store the value of the JMS message.

Value¶
Name Type Optional Default Value Documentation
messageID STRING false   This field stores the value of Message.getJMSMessageID().
messageType STRING false   This field stores the type of message that was received. This corresponds to the subinterfaces of [Message](http://docs.oracle.com/javaee/6/api/javax/jms/Message.html). [BytesMessage](http://docs.oracle.com/javaee/6/api/javax/jms/BytesMessage.html) = bytes, [MapMessage](http://docs.oracle.com/javaee/6/api/javax/jms/MapMessage.html) = map, [ObjectMessage](http://docs.oracle.com/javaee/6/api/javax/jms/ObjectMessage.html) = object, [StreamMessage](http://docs.oracle.com/javaee/6/api/javax/jms/StreamMessage.html) = stream and [TextMessage](http://docs.oracle.com/javaee/6/api/javax/jms/TextMessage.html) = text. The corresponding field will be populated with the values from the respective Message subinterface.
timestamp INT64 false   Data from the getJMSTimestamp() method
deliveryMode INT32 false   This field stores the value of Message.getJMSDeliveryMode().
correlationID STRING false   This field stores the value of Message.getJMSCorrelationID().
replyTo STRUCT true   This schema is used to represent a JMS Destination. Typically a [Queue](http://docs.oracle.com/javaee/6/api/javax/jms/Queue.html) or a [Topic](http://docs.oracle.com/javaee/6/api/javax/jms/Topic.html)
destination STRUCT true   This schema is used to represent a JMS Destination. Typically a [Queue](http://docs.oracle.com/javaee/6/api/javax/jms/Queue.html) or a [Topic](http://docs.oracle.com/javaee/6/api/javax/jms/Topic.html)
redelivered BOOLEAN false   This field stores the value of Message.getJMSRedelivered().
type STRING true   This field stores the value of Message.getJMSType().
expiration INT64 false   This field stores the value of Message.getJMSExpiration().
priority INT32 false   This field stores the value of Message.getJMSPriority().
properties Map of STRING, STRUCT false   This field stores the data from all of the properties for the Message indexed by their propertyName.
bytes BYTES true   This field stores the value from BytesMessage.html.readBytes(byte[])
map Map of STRING, STRUCT true   This field stores the data from all of the map entries returned from MapMessage.getMapNames() for the Message indexed by their key.
text STRING true   This field stores the value from [TextMessage.html.getText()](http://docs.oracle.com/javaee/6/api/javax/jms/TextMessage.html#getText()
