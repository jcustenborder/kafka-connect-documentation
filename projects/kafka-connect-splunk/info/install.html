Install¶

RPM Installation¶

Before starting the RPM installation you must configure the YUM Repository first.

sudo yum install kafka-connect-splunk

DEB Installation¶

Before starting the RPM installation you must configure the APT Repository first.

sudo apt-get install kafka-connect-splunk

Maven Installation¶

<dependency>
    <groupId>com.github.jcustenborder.kafka.connect</groupId>
    <artifactId>kafka-connect-splunk</artifactId>
    <version>0.2.0.28</version>
</dependency>
