===================
FIX Transformations
===================

Installation through the `Confluent Hub Client <https://docs.confluent.io/current/connect/managing/confluent-hub/client.html>`_.

.. code-block:: json

    confluent-hub install jcustenborder/kafka-connect-transform-fix:latest

This project provides transformations that can be utilised to process messages that are encoded with the FIX protocol.
