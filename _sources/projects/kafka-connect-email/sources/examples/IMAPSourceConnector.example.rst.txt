=====
Gmail
=====

This example will connect to an IMAP endpoint for a gmail account.



.. code-block:: json
    :caption: Configuration

    {
      "connector.class" : "com.github.jcustenborder.kafka.connect.email.IMAPSourceConnector",
      "imap.hostname" : "imap.gmail.com",
      "imap.username" : "example@gmail.com",
      "imap.password" : "osdnfgviudfngosdrt",
      "imap.folder" : "INBOX",
      "imap.topic" : "email"
    }
