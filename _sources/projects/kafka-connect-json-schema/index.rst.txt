===========
Json Schema
===========

Installation through the `Confluent Hub Client <https://docs.confluent.io/current/connect/managing/confluent-hub/client.html>`_.

.. code-block:: bash

    confluent-hub install jcustenborder/kafka-connect-json-schema:latest

This plugin is used to add additional JSON parsing functionality to Kafka Connect.
