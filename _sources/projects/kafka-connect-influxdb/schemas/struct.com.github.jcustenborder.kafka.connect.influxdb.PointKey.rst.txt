========
PointKey
========

This schema represents the :term:`key` that is be written to the Kafka containing the data.


------
Fields
------


^^^^^^^^^^^
measurement
^^^^^^^^^^^

The name of the measurement.

**Optional:** yes

**Type:** :ref:`schema-String`



^^^^
tags
^^^^

Tags associated with the measurement.

**Optional:** yes

**Type:** Map of :ref:`schema-String`, :ref:`schema-String`
