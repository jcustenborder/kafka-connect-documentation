=======================
Maxmind Transformations
=======================

Installation through the `Confluent Hub Client <https://docs.confluent.io/current/connect/managing/confluent-hub/client.html>`_.

.. code-block:: bash

    confluent-hub install jcustenborder/kafka-connect-transform-maxmind:latest

This plugin provides transformations to lookup against a MaxMind GeoIP database.
